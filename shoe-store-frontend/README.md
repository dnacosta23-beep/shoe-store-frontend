# Shoe Store Frontend

## What the App Does

This React application displays a list of shoes for sale. It retrieves shoe data from the Flask backend, allows users to add new shoes through a form, and updates the page with the new shoe after it is submitted.

## How to Run the Frontend

1. Open a terminal.
2. Navigate to the frontend project folder.
3. Install dependencies (first time only):

```bash
npm install
```

4. Start the React development server:

```bash
npm run dev
```

5. Open the URL shown in the terminal (typically):

```
http://localhost:5173
```

**Note:** The Flask backend must also be running for the application to retrieve and add shoe data.
